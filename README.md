# Nihaopay API Docs

NihaoPay API Docs compiled into markdown files, optimized for MCP workflows. 

# Overview
- `content/`: main docs content md files
- `schemas/`: zod schemas
- `scripts/`: linter and other utils

## Releases
Download releases from the github releases section. The releases tarball includes parsed and linted `content`.
